## ğŸ¯ What Each Folder Contains

### `data/raw/`
- **LinkedIN_Caption_DataSet.xlsx**: Original dataset with 2,500 LinkedIn posts
  - Columns: post_id, post_text, likes, comments, shares

### `data/processed/`
Generated by running `python src/data_processing_pipeline.py`

- **processed_data.csv**: Full dataset with 20+ features
- **train_data.csv**: 80% of data for training
- **test_data.csv**: 20% of data for testing

### `src/preprocessing/`
Data loading and cleaning modules

- **data_loader.py**: 
  - Load raw Excel data
  - Calculate engagement scores
  - Split into train/test sets
  - Save processed data

- **text_cleaner.py**:
  - Remove URLs, mentions
  - Clean whitespace
  - Extract hashtags
  - Remove emojis (optional)

### `src/features/`
Feature engineering modules

- **text_features.py** (15+ features):
  - Text statistics (length, word count, line count)
  - Punctuation (exclamations, questions)
  - Social elements (hashtags, emojis, URLs, mentions)
  - Content analysis (CTA detection, questions)
  - Capitalization metrics

- **engagement_features.py**:
  - Weighted engagement score: likes + (2Ã—comments) + (3Ã—shares)
  - Engagement ratios (comment/like, share/like, etc.)
  - Engagement intensity metrics

### `src/models/`
Saved machine learning models (generated by notebooks)

- **engagement_model.pkl**: Trained Random Forest classifier
- **tfidf_vectorizer.pkl**: Fitted TF-IDF vectorizer

### `notebooks/`
Jupyter notebooks for analysis and modeling

- **01_data_preparation.ipynb**: 
  - Data exploration
  - Statistical analysis
  - Visualization

- **02_engagement_prediction_model.ipynb**:
  - Model training (Random Forest)
  - Performance evaluation
  - Line-by-line content attribution
  - Feature importance analysis
  - Model saving

### `frontend/`
Beautiful, modern web interface with gradients and animations

- **index.html**: Home page with hero section, features, disclaimer
- **analysis.html**: Post analysis with results display
- **about.html**: Project info, model details, ethical AI practices
- **css/styles.css**: Complete styling with:
  - Purple gradient background
  - Glassmorphism effects
  - 33 animated particles (circles, squares, triangles, stars, etc.)
  - Smooth animations and transitions
  - Responsive design
- **js/script.js**: API integration, form handling, animations
